# 🛒 Army Canteen Management System

## 📌 Project Overview
This is a Java-based CRUD application developed using Core Java and JDBC.
The system allows management of products in an army canteen database.

## 🚀 Technologies Used
- Core Java
- JDBC
- PostgreSQL
- OOP Concepts

## ✨ Features
- Add Product
- Update Product
- Delete Product
- Fetch Product Details
- Database Connectivity

## 🎯 Purpose
This project is built to strengthen backend development skills and understand database operations using JDBC.
